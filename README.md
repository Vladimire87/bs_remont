# bs_remont

## Оптимизация медиа

- В репозитории оставлены только `*.webp` версии локальных изображений (включая логотипы). Старые `jpg/png` удалены, чтобы не тянуть тяжёлые файлы в продакшен.
- Если нужно добавить новые фото, сохраните исходник в `media/` или `logos/`, а затем прогоните конвертацию в WebP. Пример команды:

  ```bash
  find media logos -type f \( -iname '*.jpg' -o -iname '*.jpeg' -o -iname '*.png' \) -print0 \
    | while IFS= read -r -d '' file; do magick "$file" -strip -quality 82 -define webp:method=6 "${file%.*}.webp"; done
  ```

  После генерации замените упоминания файлов в HTML на новую `*.webp` версию и удалите исходники, чтобы не держать дубликаты.
- Все критичные картинки на страницах (`index.html`, `projects/project-1.html`, `projects/project-2.html`) теперь указывают на WebP, благодаря чему вес карточек и галерей падает в 2–4 раза.
- Для сторонних заглушек (placehold.co) ничего делать не нужно — они грузятся по сети в уже оптимизированном виде.

При необходимости поддержки браузеров без WebP можно временно вернуть fallback-изображения и обернуть `<img>` в `<picture>`, но для нынешних требований это не понадобилось.
